# PBO-E---BankSampah-Intan-Retno-Anjani-

Bank sampah adalah sistem yang dibuat menggunakan konsep objek untuk mengelola kegiatan pengumpulan sampah yang memiliki nilai ekonomi. Dalam sistem ini terdapat beberapa class seperti Nasabah, Transaksi, dan BankSampah yang memiliki atribut (data) dan method (fungsi). Nasabah dapat menyetor sampah, kemudian dicatat sebagai transaksi, dan hasilnya akan menambah saldo nasabah. Dengan menggunakan konsep PBO seperti class, object, atribut, dan method, sistem bank sampah dapat dibuat lebih terstruktur sehingga memudahkan pengelolaan data nasabah, pencatatan transaksi, dan perhitungan saldo.
